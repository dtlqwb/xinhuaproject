### API测试

使用Postman或curl测试:

```bash
# 测试登录
curl -X POST http://服务器IP:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"test","password":"123456"}'

# 测试添加客户(FormData)
curl -X POST http://服务器IP:8080/api/customer/add \
  -F "content=张三,某某公司,经理,13800138000,需要采购软件" \
  -F "salesId=1"

# 测试客户列表
curl http://服务器IP:8080/api/customer/list?salesId=1
```
