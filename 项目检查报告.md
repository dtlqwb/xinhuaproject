### 3. Reports.vue的loading状态管理问题
- **问题**: Reports.vue使用`van-list`和`van-pull-refresh`,但没有在`loadReports()`中正确控制`loading`状态
- **代码位置**: Reports.vue第173-204行
- **当前代码**:
```typescript
const loading = ref(false)

const loadReports = async () => {
  try {
    const data = await request.get('/admin/reports')
    reports.value = data || []
    finished.value = true
  } catch (error: any) {
    // ...
  }
}
```

- **问题**: `loading`变量定义了但没有在`loadReports()`中使用!
- **对比Customers.vue** (正确的实现):
```typescript
const loadCustomers = async () => {
  loading.value = true  // ← 设置loading
  try {
    const response = await request.get<any>('/admin/customers')
    customers.value = response || []
  } catch (error: any) {
    // ...
  } finally {
    loading.value = false  // ← 清除loading
  }
}
```

- **影响**: 
  1. 数据加载时没有loading动画
  2. `van-list`的`v-model:loading`绑定可能不工作
  3. 用户体验差

- **建议修复**: 在`loadReports()`中添加loading控制
