# 智销助手 - 测试指南

## ✅ 前端已启动成功

销售端H5已经成功启动在：**http://localhost:3001**

您可以点击工具面板中的预览按钮查看应用。

## 📋 测试步骤

### 1. 登录测试

**测试账号：**
- 手机号：`13800138000`
- 密码：`123456`

**预期结果：**
- 输入账号密码后点击登录
- 显示"登录成功"提示
- 跳转到主页面

### 2. 语音录入测试（需要Chrome浏览器）

**操作步骤：**
1. 登录成功后进入主页面
2. 看到底部有一个横条，显示"🎤 按住说话录入客户信息"
3. **按住横条**并开始说话，例如：
   ```
   张三，某某科技公司，销售经理，电话13800138000，需要采购办公软件
   ```
4. 松开手指
5. 系统会自动识别语音并保存到后端

**预期结果：**
- 录音时横条会变色并有放大效果
- 实时显示识别的文字
- 松开后显示"正在保存..."
- 保存成功后在上方出现一个新的客户卡片
- 卡片显示：客户姓名、公司名称

### 3. 文字输入测试

**操作步骤：**
1. 点击右侧的键盘图标 ⌨️
2. 切换到文字输入模式
3. 在输入框中输入：
   ```
   李四，YY科技有限公司，技术总监，13900139000，想了解产品详情
   ```
4. 点击"发送"按钮

**预期结果：**
- 与语音录入相同，生成新的客户卡片

### 4. 附件上传测试

**操作步骤：**
1. 点击左侧的 "+" 按钮
2. 选择图片或文档文件（可多选）
3. 然后进行语音或文字输入
4. 提交客户信息

**预期结果：**
- 显示"📎 X个附件"提示
- 客户卡片上显示附件数量

### 5. 查看客户详情

**操作步骤：**
1. 点击任意一个客户卡片

**预期结果：**
- 弹出对话框显示客户详细信息
- 包括：姓名、公司、职位、电话、需求、原始内容

### 6. 查看今日统计

**操作步骤：**
1. 点击右上角的"今日 X 个"标签

**预期结果：**
- 弹出通知显示今日已录入的客户数量

## 🔧 后端启动（可选）

如果您想测试完整的前后端交互，需要启动后端服务：

### 前提条件
1. 安装 MySQL 8.0
2. 执行数据库脚本：`backend/src/main/resources/schema.sql`
3. 安装 JDK 8+ 和 Maven

### 启动步骤

```bash
cd backend
mvn spring-boot:run
```

后端将在 http://localhost:8080 启动

**注意：** 
- 如果不启动后端，前端会显示网络错误
- 可以先体验前端界面和交互效果
- 语音识别功能在前端即可完成（浏览器支持的情况下）

## 🐛 已知限制

1. **语音识别**：仅在Chrome浏览器中支持最好，其他浏览器可能不支持
2. **后端未启动**：如果只启动前端，保存客户时会显示网络错误
3. **AI解析**：当前使用正则表达式模拟，实际应接入真实AI服务

## 📱 移动端测试

在手机上访问：
1. 确保手机和电脑在同一WiFi下
2. 启动时添加 `--host` 参数：
   ```bash
   npm run dev -- --host
   ```
3. 使用手机浏览器访问显示的IP地址
